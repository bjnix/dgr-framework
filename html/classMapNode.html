Detailed Description

template<typename T>
class MapNode< T >

MapNode Class Template that catalogues and serializes an object of any type as well as take in a raw char* representation of that data and parse it back into the correct format

Attention
only works with POD types without padded data (i.e no structs, unions) if an object is not guaranteed to have contiguous data, YOU MUST specialize the template in order to use it
